## Setup

```bash
pip install -r requirements.txt
```

## Phase 1: Environment Setup & Exploration

Run the initial exploration:

```bash
python src/phase1_environment_setup.py
```

This will:

- Load the Space Invaders environment
- Run a random agent
- Display environment statistics
- Save sample frames
